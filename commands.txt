COMANDOS GO
go run {PATH_FILE} - Executa o código (go run *.go para compilar e rodar projetos com vários arquivos)
go fmt {PATH_FILE} - Formata o arquivo
got build - Compila o código

COMANDOS DEP
dep init - Inicia um projeto com dep
dep ensure - Atualiza o projeto com as dependências adicionados no arquivo Gopkg.toml

COMANDOS REALIZE
realize start - Cria o arquivo de configuração .realize.yaml.
realize add - Adiciona o projeto ao arquivo .realize.yaml.
realize init - Realiza a configuração do .realize.yaml.
